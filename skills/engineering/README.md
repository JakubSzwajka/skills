# Engineering Skills

Engineering skills are grouped by invocation shape:

- `agents/` — explicit-use engineering decision personas and their role-scoped helper skills.
- `quality/` — review and test helper workflows used by default execution.
- `docs/` — documentation, PRD, research, repo-knowledge, and task-harvest workflows.
- `commit/` — explicit commit workflow.

Rule of thumb: invoke an agent when you want a product, architecture, or design decision lens; invoke a docs/quality/commit skill when you want a bounded process to run.
