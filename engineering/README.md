# Engineering Skills

Engineering skills are grouped by invocation shape:

- `agents/` — explicit-use engineering personas and their role-scoped helper skills.
- `docs/` — documentation, PRD, research, and repo-knowledge workflows.
- `commit/` — explicit commit workflow.

Rule of thumb: invoke an agent when you want a role/lens; invoke a docs/commit skill when you want a bounded process to run.
