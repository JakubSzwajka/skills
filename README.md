# Agent Skills

A collection of reusable skills for Claude Code. All custom skills use the `jsz-` prefix.

## Skills

| Skill | Description |
|-------|-------------|
| **jsz-commit** | Analyze changes, run checks, and commit with a clear message |
| **jsz-create-skill** | Guided discovery and generation of new skills |
| **jsz-find-skills** | Discover and install agent skills |
| **jsz-memory** | Session memory reflection and continuity |
| **jsz-modularity** | Capabilities-vs-operations modularity analysis |
| **jsz-pipeline** | Execute PRD tasks in dependency-ordered parallel waves |
| **jsz-prd-challenge** | Stress-test PRDs against the real codebase |
| **jsz-prd-create** | Create lightweight PRDs and decompose into tasks |
| **jsz-research** | Deep codebase exploration with structured reports |
| **jsz-review** | Pre-push code review for correctness and production readiness |
| **jsz-tdd-fix** | Fix triaged bugs using test-driven development |
| **jsz-test** | Run tests and produce focused, actionable reports |
| **jsz-triage** | Investigate and triage bug reports with structured output |
| **jsz-update-docs** | Keep module README files in sync with code changes |
