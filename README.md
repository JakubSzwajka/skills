# Agent Skills

A collection of reusable skills for Claude Code.

## Skills

| Skill | Description |
|-------|-------------|
| **adr-skill** | Create and maintain Architecture Decision Records |
| **create-skill** | Guided discovery and generation of new skills |
| **deep-research** | Deep codebase exploration with structured reports |
| **find-skills** | Discover and install agent skills |
| **memory** | Session memory reflection and continuity |
| **modularity-review** | Capabilities-vs-operations modularity analysis |
| **pipeline** | Execute PRD tasks in dependency-ordered parallel waves |
| **prd-challenger** | Stress-test PRDs against the real codebase |
| **readme-update** | Keep module README files in sync with code changes |
| **smart-commit** | Analyze changes, run checks, and commit with a clear message |
| **smart-prd** | Create lightweight PRDs and decompose into tasks |
| **smart-review** | Pre-push code review for correctness and production readiness |
| **test-runner** | Run tests and produce focused, actionable reports |
| **tdd-fix** | Fix triaged bugs using test-driven development |
| **triage** | Investigate and triage bug reports with structured output |
